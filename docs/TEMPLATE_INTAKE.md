# PROJECT INTAKE

## Project Name
English Learning System

## Primary Goals
- Track vocabulary learned from DailyDictation
- Automatically analyze grammar using AI
- Use spaced repetition to review vocabulary
- Build a personal English learning system

## Target Users
Phase 1: Personal use  
Phase 2: Potential public SaaS for English learners

## Current Learning Workflow
1. Listen to the audios
2. Type the sentence
3. Check the correct answer
4. Identify new vocabulary

## Problem Statement
DailyDictation does not:
- track learned vocabulary
- explain grammar
- provide spaced repetition review

## Desired Solution
A companion system that:
- captures sentences
- analyzes vocabulary and grammar with AI
- stores vocabulary
- schedules spaced repetition reviews

## Preferred Tech Stack
Backend:
- NestJS
- PostgreSQL
- Redis

Frontend:
- React / Vite

AI:
- OpenAI API

Browser Integration:
- Chrome Extension

## MVP Scope
Option B:
- Chrome Extension
- AI sentence analysis
- Vocabulary tracking
